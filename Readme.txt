# TechConnect 2025 🎉

Uma landing page responsiva e moderna para um evento fictício de tecnologia. Criado com HTML, CSS e JavaScript puro, com foco em design acessível, boa semântica e validação de formulário.

![Capa do Projeto](img/capa.png) <!-- Substitua ou remova se não tiver imagem -->

## 📌 Sobre o Projeto

**TechConnect 2025** é um projeto de exemplo de uma landing page para eventos, ideal para freelancers, conferências ou empresas. Desenvolvido com atenção a:

- ✅ Acessibilidade (uso de aria-labels, landmarks e contraste adequado)
- ✅ Layout moderno e responsivo
- ✅ Animações com AOS.js
- ✅ Validação real de formulário com JavaScript

---

## 🚀 Tecnologias Utilizadas

- HTML5 semântico
- CSS3 moderno e responsivo
- JavaScript puro (sem frameworks)
- [AOS.js](https://michalsnik.github.io/aos/) para animações ao rolar a página
- Google Fonts (`Inter`)

---

## 📷 Seções do site

- Hero com chamada para ação
- Sobre o evento
- Cronograma
- Palestrantes
- Localização com Google Maps
- Formulário de inscrição funcional
- Rodapé com redes sociais

---

## 💡 Funcionalidades

- Validação de e-mail e nome no formulário
- Feedback de erro e sucesso via alert
- Estilo responsivo adaptado para mobile
- Preparado para deploy no GitHub Pages ou Netlify
